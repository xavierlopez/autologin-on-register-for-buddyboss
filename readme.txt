== Description ==

This plugin simplifies register process by removing the First Name and using nickname instead on BuddyBoss Platform.


Requirements:
BuddyBoss Platform plugin must be active.

== Installation ==
1. Upload the entire 'bb-autologin-on-register' folder to the `/wp-content/plugins/` directory.
1. Make sure 'BuddyBoss Platform' plugin is active before activating this plugin.
1. Activate the plugin through the 'Plugins' menu in WordPress.
